We would like to point out the following limitations:

  • Processing time increases significantly when balancing/transferring ships with >30 containers
  • SIFT takes a significant amount of processing times for ships with more than 8 containers
  • Manifest Upload is not automatically asked for but is forced in the home page
  • Language selector was not implemened
