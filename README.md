## Usage

```
git clone https://github.com/andykr1k/pitech.git
cd PiTech
```

### Installation

#### Create & Start Virtual Environment

Windows
```
python -m venv env
source env/Scripts/activate
```

Linux
```
python -m venv env
source env/bin/activate
```

#### Install Requirements

```
pip install -r requirements.txt
```

### Run Platform

```
python main.py
```
